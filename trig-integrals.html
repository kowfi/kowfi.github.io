Step 3: Isolate x, dx, and f(x) 

Case 1: √(a²-x²)
     x = a/x*sinθ
    dx = a/x*cosθ     
  f(x) = a*cosθ

Case 2: √(a²+x²)
     x = a/x*tanθ
    dx = a/x*sec²θ 
  f(x) = a/cosθ, or √(a²+x²) = a*secθ

Case 3: √(x²-a²)
     x = a/x*cosθ, 
       = a*secθ
    dx = a/x*sec²θ 
  f(x) = a/cosθ, or √(x²-a²) = a*secθ

Integration of Trigonometric Function 

Case 1: n = Odd
  a) ∫sinnxdx
   = ∫(sinn-1x)sinxdx
       sin2x=1-cos2x

  b) ∫cosnxdx
   = ∫(cosn-1x)cosxdx
       cos2x=1-sin2x

Case 2: either n or m is Odd/Even, basta may isang Odd
  ∫sinncosmxdx
     i) n = +Odd int.
      = ∫sinn-1xcosmxsinxdx
      = ∫(sin2x)n-1/2xcosmxsinxdx
          sin2x=1-cos2x

     ii) m = +Odd int.
      = ∫sinnxcosm-1xcosxdx
      = ∫sinn(cos2x)m-1/2xcosxdx
              cos2x=1-sin2x

Case 3: n = Even, m = Even
  a) ∫sinnxdx
   = ∫(sin2x)n/2dx
       sin2x=1-cos2x/2
  b) ∫cosnxdx
   = ∫(cos2x)n/2dx
       cos2x=1+cos2x/2

  c) ∫sinnxcosmxdx
   = ∫(sin2x)n/2(cos2x)m/2dx


Case 4: n = Odd or Even
  a) ∫tannxdx
   = ∫tann-2xtan2xdx
             tan2x=sec2x-1

  b) ∫cotnxdx
   = ∫cotn-2xcot2xdx
             cot2x=csc2x-1

Case 5: n = Even
  a) ∫secnxdx
   = ∫secn-2xsec2xdx
   = ∫(sec2x)n-2/2sec2xdx
       sec2x=tan2x+1

  b) ∫cscnxdx
   = ∫cscn-2xcsc2xdx
   = ∫(csc2x)n-2/2csc2xdx
       csc2x=1+cot2x

Case 6: n = Odd/Even, m = Even
  a) ∫tannxsecmxdx
   = ∫tannxsecm-2xsec2xdx
   = ∫tannx(sec2x)m-2/2sec2xdx
            sec2x=tan2x+1

  b) ∫cotnxcscmxdx
   = ∫cotnxcscm-2xcsc2xdx
   = ∫cotnx(csc2x)m-2/2csc2xdx
            csc2x=1+cot2x

Case 7: n = Odd, m = Odd
  a) ∫tannxsecmxdx
   = ∫tann-1xsecm-1xsecxtanxdx
   = ∫(tan2x)n-1/2secm-1xsecxtanxdx
      tan2x=sec2x-1

  b) ∫cotnxcscmxdx
   = ∫cotn-1xcscm-1xcscxcotxdx
   = ∫(cot2x)n-1/2cscm-1xcscxcotxdx
      cot2x=csc2x-1

Case 8: n = Odd
  a) ∫secnxdx = ∫secn-2xsec2xdx
               u=secn-2x  dv=sec2xdx

  b) ∫cscnxdx = ∫cscn-2xcsc2xdx
               u=cscn-2x  dv=csc2xdx

Case 9: n = Even, m = Odd
  a) ∫tannxsecmxdx
   = ∫(tan2x)n/2secmxdx
       tan2x=sec2x-1

  b) ∫cotnxcscmxdx
   = ∫(cot2x)n/2cscmxdx
       cot2x=csc2x-1
